# Vue Applications Manager

A frontend landing page that displays a collection of Vue.js projects as cards with images, descriptions, and links to live demos.

### Live Demo on Firebase: <https://app.karnaukh-webdev.com>

![Apps Manager screenshot](https://raw.githubusercontent.com/SerhiiKarnaukh/vue-test-manager/main/src/apps/apps_manager/vue_manager.jpg)

## Architecture

```
src/apps/apps_manager/
├── README.md
├── vue_manager.jpg
├── routes.js
├── api/vueApps.js
├── components/                   # TheNavbar, TheFooter, TheAppCard
├── layouts/MainAppsManagerLayout.vue
└── views/                        # Home, Search, NotFound
```
